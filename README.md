## Discord music bot
Discord music bot using `discord.js`

* !play command.
* !np - now play
* !pause
* !queue
* !resume
* !skip
* !stop
* !volume

### Start bot server
```sh
node src/index.js
```

### Search on youube and play in voice channel
> !play justin bieber baby

### Install nodejs & npm
```sh
curl -sL https://deb.nodesource.com/setup_15.x | sudo -E bash -
sudo apt-get install -y nodejs
node -v
npm -v
```
